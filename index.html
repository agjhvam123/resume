  • 系统稳定性优化：引入 Watchdog 守护进程，对相机断连、网络抖动场景进行异常恢复；通过 Valgrind 消除内存泄漏，长跑测试 72h 无异常。
  • Uart实时接收秤数据：采用 ringbuffer + 协议状态机处理流式数据，解决半包/粘包及错位重同步问题；

FreeRTOS PWM DMA 时钟树 OTA
▶ 展开技术细节（6 项核心实现） ▾
  • 调度策略设计：开启移抢占式调度，构建“采集→算法→通信”实时链路，按实时性划分优先级；对计算任务做分段/让出 CPU，结合同级时间片轮转，保证采样与通信在高负载下仍可及时响应。
  • 固件升级：采用了Bootloader + Application 双区架构，划分固件入口与升级区。在 OneNet 平台下搭建好 MQTT 和 HTTP 服务器，然后在MCU 实现 OTA 远程升级和 IAP 在应用升级的功能。外挂了存储器 AT24C02 和 W24Q64 ，可以同时支持版本的更新和回退。
